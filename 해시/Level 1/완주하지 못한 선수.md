## 스택 / 큐

### 나의 풀이
```javascript
function solution(participant, completion) {
    let p = participant;
    let c = completion;

    p.sort();
    c.sort();

    for(let i = 0; i < c.length; i++) {
        if(p[i] !== c[i]) return p[i];
    }

    return p[p.length - 1];
}
```
